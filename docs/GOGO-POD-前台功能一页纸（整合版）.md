## 1. 登录


## 一句话

用户用账号密码进入 GOGO POD；退出后回到登录页。

## 解决什么问题

控制谁能使用平台，并在会话过期或账号停用时给出明确提示。

## 入口

- 未登录访问任意前台页面 → 跳转登录页
- 已登录 → 左侧菜单底部 / 头像菜单 → **退出登录**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 账号输入框 | 邮箱或用户名 |
| 密码输入框 | 掩码显示 |
| 登录按钮 | 主操作 |
| 错误提示区 | 校验失败时在表单上方或下方显示 |

---

## 按钮与交互

| 操作 | 点击后发生什么 |
|------|----------------|
| **登录** | 校验账号密码；成功进入 **首页/工作流**；失败显示对应提示 |
| **退出登录** | 清除登录状态，跳转登录页 |

---

## 规则与提示文案

| 情况 | 用户看到 |
|------|----------|
| 密码错误 | 「账号或密码不正确」 |
| 账号被停用 | 「账号已停用，请联系管理员」 |
| 登录过期（Token 失效） | 「登录已过期，请重新登录」 |

---

## 与其他模块的关系

- 登录成功后，所有前台模块（作图、任务中心、我的空间等）才可用
- 后台 **用户账号** 停用某用户 → 该用户无法登录或会被强制退出

## 后台配置影响

| 后台操作 | 前台影响 |
|----------|----------|
| 停用用户 | 登录失败或使用中被迫退出 |
| 重置密码 | 用户需用新密码登录 |

## 2. 首页与工作流


## 一句话

把多个 AI 步骤串成一条流水线，选一批图一键跑完（如抠图 → 裂变 → 套图）。

## 解决什么问题

避免在 8 个作图功能之间来回切换；适合批量、标准化处理。

## 入口

左侧菜单 → **首页 / 工作流**

---

## 页面长什么样

### 工作流首页

| 区域 | 内容 |
|------|------|
| 欢迎 / 快速开始 | 引导新建或从模板开始 |
| 最近任务 | 最近跑过的工作流及状态 |
| 推荐玩法 / 案例 | 运营配置的示例卡片（来自后台「推荐案例」） |

### 工作流编辑器（画布）

| 区域 | 内容 |
|------|------|
| 画布 | 节点与连线 |
| 节点面板 | 可添加的节点类型列表 |
| 右侧参数面板 | 选中 AI 节点后出现，调该步参数 |
| 顶部操作栏 | 保存、运行、返回等 |

### 模板库

| 分类 | 说明 |
|------|------|
| 官方模板 | 后台下发，全员可见（已发布） |
| 我的模板 | 用户自己保存的流程 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建工作流** | 打开空白画布 |
| **从模板创建** | 选模板 → 生成可编辑的工作流副本 |
| **添加节点** | 节点出现在画布上 |
| **连线** | 拖拽连接两节点，定义执行顺序 |
| **删除节点 / 连线** | 该步骤不再执行 |
| **保存** | 保存当前流程，下次可继续编辑 |
| **运行 / 开始** | 选择输入图片 → 提交任务 → 去 **任务中心** 看进度 |
| **另存为模板**（若开放） | 存为「我的模板」 |
| **使用 / 创建**（模板卡片） | 用该模板生成新工作流 |
| **预览**（模板卡片） | 查看模板包含哪些步骤 |

---

## 规则

- 流程必须能从「输入」走到「输出」，否则运行前应提示不完整
- 后台停用某 AI 功能 → 画布中对应 **节点不可选或置灰**，已有流程中含该节点时运行应提示
- 运行中的任务在 **任务中心** 查看；单步失败可看到是哪一步、什么原因

---

## 与其他模块的关系

| 模块 | 关系 |
|------|------|
| 我的空间 / 找图 | 可作为工作流输入来源 |
| 作图 | 工作流中的 AI 节点对应各作图能力 |
| 任务中心 | 运行结果、进度、重试 |
| 下载中心 | 若输出节点配置为下载 |

## 后台配置影响

| 后台模块 | 影响 |
|----------|------|
| AI 功能配置 | 某功能关闭 → 节点不可用 |
| 场景预设 | 节点内参数面板与单功能作图一致 |
| 工作流模板 | 官方模板增删改 |
| 推荐案例 | 首页案例卡片 |

## 3. 找图


## 一句话

从外部电商网站采集图片到平台，或用一张图去外部网站搜相似图。

## 解决什么问题

POD 卖家素材来源在外部平台，需要先把图「抓进来」再作图。

## 入口

左侧菜单 → **找图**

---

## 页面长什么样

### A. 插件采集（采集任务列表）

| 列 / 信息 | 说明 |
|-----------|------|
| 采集来源 | 网站 / 店铺 |
| 采集时间 | — |
| 图片数量 | 本次采了多少张 |
| 状态 | 同步中 / 完成 / 失败 |

### B. 以图搜图

| 区域 | 说明 |
|------|------|
| 上传 / 选图 | 本地或从我的空间 |
| 外部站点按钮 | Google、Pinterest、Shutterstock 等 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **查看详情** | 展开该次采集的全部图片 |
| **批量选择** | 进入多选模式 |
| **入库到我的空间** | 选中图保存到 **我的空间** |
| **发送到作图** | 带着选中图打开 **作图** 模块（需再选具体功能） |
| **发送到工作流** | 选中图作为工作流输入 |
| **去 Google / Pinterest / … 搜** | **新开浏览器标签页**，跳转外部站以图搜图（不在站内完成） |

---

## 规则

- 采集任务失败应显示原因，支持重试（若产品支持）
- 以图搜图依赖外部网站，站内只负责跳转
- 发送到作图 / 工作流时，若未选图应提示先选择

---

## 与其他模块的关系

| 模块 | 关系 |
|------|------|
| 我的空间 | 采集结果入库 |
| 作图 / 工作流 | 采集图作为输入 |

## 后台配置影响

一般无强依赖；若未来做「禁止某些外部站点入口」，由 **系统设置** 控制。

## 4. 印花提取


## 一句话

从商品实拍图里把印花/图案单独抠出来，常见输出透明底 PNG。

## 解决什么问题

商品图上有衣服、背景等干扰，卖家只需要中间的图案去套图或再创作。

## 入口

左侧菜单 → **作图** → **印花图提取**（或从找图/我的空间「发送到作图」带入图片）

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 上传 / 从我的空间选 / 拖拽 |
| 参数区 | 由后台「场景预设」配置（见下表典型项） |
| 底部 | 取消、**提交** |

---

## 典型参数（以场景预设为准）

| 参数 | 用户理解 | 交互 |
|------|----------|------|
| 模式选择 | 专项提取 vs 全能提取 | 单选；专项适合遮挡少的印花类，全能适合褶皱、低清、全副印花 |
| 透明底图 | 输出是否透明背景 | 开关 |
| 分辨率 | 标清(1k) / 超清(4k) | 单选 |
| 尺寸比例 | 1:1、2:3 等 | 下拉 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **取消** | 关闭弹窗，不提交 |
| **提交** | 创建任务 → **任务中心** 看进度；完成后可预览结果 |
| **入库**（结果页） | 存到 **我的空间** |
| **下载** | 打包下载，记录进 **下载中心** |
| **再创作** | 用输出图作为输入跑其他功能（如套图） |
| **对比查看** | 原图 vs 提取结果 |
| **重置参数** | 恢复后台配置的默认值 |

---

## 规则

- 未上传素材时 **提交** 应禁用或提示
- 参数默认值来自后台 **场景预设 → 印花图提取 → 默认**
- 后台关闭本功能 → 入口隐藏或置灰

---

## 典型后续流程

提取出的印花 → **商品套图** → 入库 / 上架

## 后台配置影响

| 配置项 | 前台变化 |
|--------|----------|
| AI 功能配置 · 停用 | 入口不可用 |
| 场景预设 · 控件/默认值 | 参数项、默认值、提示词效果 |

## 5. 一键抠图


## 一句话

把图片里的主体从背景中分离出来，得到可透明底或纯色底的素材。

## 解决什么问题

商品、人物、图案需要干净主体，用于裂变、套图或合成。

## 入口

左侧菜单 → **作图** → **一键抠图**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 上传 / 从我的空间选 |
| 参数区 | 抠图模式、边缘处理等 |

---

## 典型参数

| 参数 | 说明 | 交互 |
|------|------|------|
| 抠图模式 | 去背景 / 抠头 | 单选，带帮助说明 |
| 边缘处理 | 自动裁掉印花边缘空白 | 开关；开启后边缘更干净 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **取消** | 关闭，不提交 |
| **提交** | 创建抠图任务 → **任务中心** |
| **入库 / 下载 / 再创作 / 对比** | 与其他作图功能一致 |

---

## 规则

- 至少上传 1 张图才能提交
- 参数由 **场景预设 → 一键抠图 → 默认** 决定

---

## 典型后续流程

抠图结果 → **图裂变** / **商品套图** / **我的空间**

## 后台配置影响

AI 功能开关 + 场景预设（参数面板）

## 6. 图裂变


## 一句话

基于一张参考图，批量生成多张风格或细节不同的变体图（平台核心能力）。

## 解决什么问题

同一爆款需要多个差异化版本测款、铺货，避免手工改图。

## 入口

左侧菜单 → **作图** → **图裂变**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| **场景 Tab**（顶部） | 默认、服装/纺织、手机壳、铁艺图形、挂钟、装饰画、铁皮画等 |
| 素材上传区 | 参考图（可多张，视产品限制） |
| 参数区 | 模式、强度、比例、出图数量等 |
| 底部 | 取消、提交 |

---

## 典型参数与交互

| 参数 | 说明 | 交互规则 |
|------|------|----------|
| 模式选择 | 艺术设计 / 文字强化 / 爆款二创 / 通用 | 单选；**选中不同模式会展开不同子参数** |
| 原图参考强度 | 多像原图 | 滑条 0~1 |
| 创意发散强度 | 创意程度 | 滑条，可显示「低/中/高」 |
| 指定背景色 | 随机 / 黑 / 白 | 单选（文字强化模式下） |
| 裂变内容 | 改主体/姿势/背景/爆改 | 单选（爆款二创模式下） |
| 形状 | 默认 / 圆形 | 单选 |
| 尺寸比例 | 原图比例、1:1 等 | 下拉 |
| 出图数量 | 1 / 2 / 4 / 6 / 8 | 数字按钮 |

**特殊场景（部分仍为专用 UI）：**

- **铁艺图形**：图形风格卡片、变化维度、专用上传区
- **挂钟**：生成模式、表盘刻度样式
- **铁皮画**：模式与特效选项

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 场景 **Tab** | 切换场景，参数区刷新为该场景预设 |
| **提交** | 按当前场景+参数创建任务 |
| 结果区 **入库 / 下载 / 再创作** | 与其他作图功能一致 |

---

## 规则

- 切换场景 Tab → 参数恢复该场景默认值
- 切换「模式选择」→ 隐藏旧模式子参数，显示新模式子参数
- 出图数量影响任务消耗与结果张数
- 后台新增/下线场景 Tab → 顶部 Tab 增删

---

## 典型后续流程

裂变结果 → 挑选满意变体 → **商品套图** / **我的空间** / **上架**

## 后台配置影响

| 配置 | 影响 |
|------|------|
| 场景预设 · 各场景 Tab | Tab 列表、每 Tab 下控件与子控件 |
| 场景预设 · prompt | 生成效果与文案 |
| AI 功能配置 · 停用 | 入口不可用 |

## 7. 文生图


## 一句话

输入文字描述（提示词），AI 直接生成设计图；支持多条 prompt 批量出图。

## 解决什么问题

没有现成素材时，从创意文字快速出稿。

## 入口

左侧菜单 → **作图** → **文生图**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 添加方式 | 图反推提示词、提示词助手、导入 Excel 等 |
| 模式 / 参数区 | 模式、比例、出图数量（来自场景预设） |
| 提示词列表 | 多条 prompt，可勾选、增删 |
| 底部 | 取消、提交 |

---

## 典型参数

| 参数 | 说明 |
|------|------|
| 模式选择 | 艺术设计 / 文字强化 / IP 增强 / 四方连续图 |
| 尺寸比例 | 1:1、2:3 等 |
| 出图数量 | 通常 2 或 4 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **图反推提示词** | 上传参考图 → 自动生成 prompt 填入列表 |
| **提示词助手** | 打开辅助写词（跳转或侧栏） |
| **导入 EXCEL** | 批量导入多条 prompt |
| **+ 添加新的提示词** | 列表增加一行输入框 |
| 行内 **删除** | 移除该条 prompt |
| 行内 **勾选** | 控制该条是否参与提交 |
| **提交** | 对「已勾选且有内容」的 prompt 批量生成 |

---

## 规则

- 建议提示使用英文（页面可有说明文案）
- 至少 1 条有效 prompt 才能提交
- 出图数量 × 有效 prompt 条数 = 总任务量（按产品设计计费/限额）
- 提示词列表为前台专用 UI，模式/比例/数量由 **场景预设** 配置

---

## 典型后续流程

生成图 → **入库** → **套图** / **上架**

## 后台配置影响

场景预设（模式、比例、数量默认值）+ AI 功能开关

## 8. 商品套图


## 一句话

把印花合成到商品样机底图上，一步生成可上架的多角度/多场景商品图。

## 解决什么问题

卖家有图案但没有商品实拍，需要快速出「穿在身上/放在桌上」的效果图。

## 入口

左侧菜单 → **作图** → **商品套图**

---

## 页面长什么样（两步向导）

### 第一步 · 模板选择

| 区域 | 内容 |
|------|------|
| 品类 Tab | 推荐、服饰、铁皮画、家用纺织、挂钟、手机壳等 |
| 模板卡片网格 | 缩略图、名称、「官方」标签、多选框 |
| 底部 | 取消、**下一步** |

### 第二步 · 印花选择

| 区域 | 内容 |
|------|------|
| 添加素材 | 上传 / 从我的空间选 |
| 印花处理参数 | 查重、边缘、描边、压缩、格式等（场景预设） |
| 底部 | 取消、**提交** |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 品类 **Tab** | 筛选模板列表 |
| 点击模板卡片 | 勾选 / 取消勾选（可多选） |
| **下一步** | 进入第二步（可从我的空间带入素材时直接进第二步） |
| **提交** | 对选中模板 + 印花创建套图任务 |
| **取消** | 关闭向导 |

---

## 典型参数（第二步 · 印花处理）

| 参数 | 说明 |
|------|------|
| 印花查重校验 | 是否检测重复印花 |
| 边缘处理 | 裁掉边缘空白 |
| 描边 | 开关；可选智能/强制、颜色、粗细 |
| 智能反转 | 自动反转颜色适配底图 |
| 压缩图片 | 限制输出文件大小 |
| 文件格式 | JPEG / PNG / WEBP |

**交互规则：** 开启「描边」才出现描边方式；选「强制描边」才出现颜色选项。

---

## 规则

- 第一步至少选 1 个模板才能下一步（建议）
- 第二步至少 1 张印花才能提交
- 模板列表来自后台 **套图模板**；停用的模板不展示
- 印花处理参数来自 **场景预设 → 商品套图 → 印花处理**

---

## 典型后续流程

## 10. 侵权风险过滤


## 一句话

检测图片是否可能侵犯商标、IP 或含敏感元素，给出风险等级与上架建议。

## 解决什么问题

POD 卖家批量铺货时避免侵权下架、账号风险。

## 入口

左侧菜单 → **作图** → **侵权风险过滤**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 待检测图片 |
| 提示条 | 建议先「图案裁切」再检测（特殊构图时） |
| 模式选择 | 深度过滤 / 基础过滤 |
| 合规勾选 | 「侵权风险过滤功能使用须知」 |
| 底部 | 取消、提交 |

---

## 典型参数

| 模式 | 用户理解 |
|------|----------|
| **深度过滤** | 结合 TRO 案件、艺术家版权库、全网检索，出证据链式报告 |
| **基础过滤** | 快速比对知名商标与热门 IP |

---

## 结果页用户看到什么

| 信息 | 说明 |
|------|------|
| 风险等级 | 低 / 中 / 高 |
| 命中原因 | 大白话说明（如疑似某品牌 logo） |
| 建议 | 可用 / 建议复核 / 不建议上架 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **示例**（提示条内） | 查看何种图建议先裁切 |
| **提交** | 创建检测任务（建议需勾选须知） |
| 结果 **复制报告** | 复制结论供运营留存 |

---

## 规则（重要）

- **高风险** 建议：默认 **不允许直接进入上架流程**，或必须二次确认
- 中风险：提示复核，用户自行决定
- 低 risk：可正常后续作图/上架
- 深度模式耗时可能更长，应在任务中心展示预计时间（若支持）

---

## 与其他模块的关系

- 检测通过/复核后 → **商品库** / **上架**
- 与 **印花提取** 组合：先提取图案再检测更准

## 后台配置影响

AI 功能开关 + 场景预设（模式选项）

## 12. 视频生成


## 一句话

## 页面长什么样

## 规则

## 13. 上架


## 一句话

把作图结果整理成「商品」，绑定店铺，按平台模板导出刊登包。

## 解决什么问题

作图完成后还要填标题、选平台格式、打包下载才能上架，需要一站式出口。

## 入口

左侧菜单 → **上架**（含商品库、店铺管理、刊登模板子页）

---

## 子功能一：商品库

### 页面长什么样

| 区域 | 内容 |
|------|------|
| 商品列表 | 主图缩略图、标题、状态 |
| 筛选 / 搜索 | 按状态、标签等 |

### 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建商品** | 从我的空间 / 任务结果选图，填标题、描述、标签 |
| **编辑** | 改标题、换主图、增删套图、改标签 |
| **删除** | 删除商品（需确认） |
| **导出** | 按选定平台模板导出 |

### 常见字段

标题、描述、主图、套图多图、标签/关键词、类目（视平台）

---

## 子功能二：店铺管理

| 按钮 | 点击后发生什么 |
|------|----------------|
| **绑定店铺** | 填写平台授权信息（本期可占位流程） |
| **解绑** | 解除关联，需确认 |

---

## 子功能三：刊登模板

**流程：**

1. 选择平台（Temu / Amazon / Shein …）
2. 选择该平台的 **刊登模板**
3. 选择要导出的商品
4. 点 **导出刊登包**

**导出结果：** 压缩包，内含图片 + 表格或字段文件（字段由后台模板定义）

---

## 规则

- 必填字段未填时导出应拦截并提示缺什么
- **侵权高风险** 商品导出前应警告或拦截（与侵权过滤联动）
- 刊登模板字段、命名规则由后台 **刊登模板 / 套图模板** 配置

---

## 与其他模块的关系

| 来源 | 关系 |
|------|------|
| 任务中心 / 我的空间 | 选图创建商品 |
| 标题提取 | 填充标题 |
| 下载中心 | 导出大包记录 |

## 后台配置影响

平台列表、刊登模板字段、导出规则、套图模板

## 15. 任务中心


## 一句话

所有 AI 与工作流任务的进度、结果、失败原因和后续操作的统一入口。

## 解决什么问题

任务异步执行，用户需要知道「跑完没有、失败为什么、结果在哪」。

## 入口

左侧菜单 → **任务中心**

---

## 页面长什么样

### 列表页

| 列 | 内容 |
|----|------|
| 任务类型 | 抠图、裂变、套图、工作流、视频… |
| 状态 | 排队中 / 处理中 / 成功 / 失败 / 已取消 |
| 创建时间 | — |
| 摘要 | 张数、场景名等 |

**筛选：** 类型、状态、时间范围

### 详情页

| 区域 | 内容 |
|------|------|
| 输入图 | 本次用的素材 |
| 参数摘要 | 用人话展示（如「模式：文字强化，出图 4 张」） |
| 输出结果 | 网格预览，可点大图 |
| 失败信息 | 原因 + 是否可重试 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 点击列表行 | 进入详情 |
| **查看结果 / 对比** | 输入 vs 输出对比视图 |
| **入库** | 输出存到 **我的空间**（或商品库） |
| **下载** | 单个或打包下载 → **下载中心** 有记录 |
| **重试** | 用相同输入+参数再跑一次（失败且允许时） |
| **再创作** | 用输出作为新任务输入（如裂变 → 套图） |
| **取消**（若支持） | 排队中任务取消 |

---

## 规则

| 状态 | 用户能做什么 |
|------|--------------|
| 排队中 | 可取消（若支持） |
| 处理中 | 仅查看，不可改参数 |
| 成功 | 入库、下载、再创作 |
| 失败 | 看重试；后台关闭的功能不可重试并说明原因 |
| 已取消 | 可重新从原模块发起 |

---

## 与其他模块的关系

所有作图、视频、工作流任务汇聚于此。

## 后台配置影响

| 操作 | 影响 |
|------|------|
| 场景预设变更 | 仅影响 **新任务** 默认值，不改变历史任务展示 |
| 功能停用 | 重试按钮不可用 |
| 任务监控 · 后台重试/取消 | 用户侧状态同步更新 |
