## 1. 概览


## 一句话

管理员一眼看平台运行是否正常，并从预警跳到对应配置页处理。

## 给谁用

运营负责人、管理员。

## 入口

管理后台登录 → 左侧 **概览**

---

## 页面长什么样

| 区域 | 典型内容 |
|------|----------|
| 数据卡片 | 用户数、今日任务量、成功率 |
| 趋势图 | 任务量 / 失败率走势（可选） |
| 预警列表 | 未配 API Key、某功能失败率高、存储异常等 |
| 快捷入口 | 跳到 AI 功能配置、任务监控等 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **刷新** | 重新加载最新统计数据 |
| **去处理**（预警卡片） | 跳转到对应模块（如补 Key、看失败任务） |

---

## 规则

- 概览 **不直接改前台**，只读 + 跳转
- 预警规则由产品定义（如：24h 失败率 > X% 标红）

---

## 对前台的影响

间接：运营根据预警修配置后，前台可用性和成功率改善。

---

## 2. AI 功能配置


## 一句话

管每个 AI 功能 **开不开**、基础信息、模型 ID 和 API 密钥。

## 给谁用

AI 配置人员、管理员。

## 入口

管理后台 → **AI 功能配置**

---

## 与「场景预设」的分工

| 本模块 | 场景预设 |
|--------|----------|
| 功能总开关 | 前台参数面板长什么样 |
| model_id、密钥 | 有哪些 Tab、滑条、默认值 |
| 功能名称、说明 | 提示词片段 |

**不要在本模块配表单控件。**

---

## 页面长什么样

功能列表，每行包含：

- 功能名称（抠图、印花提取、裂变、文生图、套图、矢量、侵权、视频、标题提取…）
- **启用 / 停用** 开关
- 简短说明
- 最近更新时间

点进某功能 → 详情页：

| 区域 | 内容 |
|------|------|
| 概览 | 名称、说明、启用状态、model_id |
| 模型与密钥 | provider、API URL、API Key |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **启用 / 停用** | 切换功能是否对前台可见可用（需保存） |
| **保存** | 写入配置；**未保存离开应提示** |
| **取消 / 重置** | 放弃未保存修改 |

---

## 对前台的影响

| 操作 | 用户看到 |
|------|----------|
| 关闭「抠图」 | 作图无抠图入口；工作流无抠图节点 |
| 开启「视频」 | 左侧出现视频菜单 |
| Key 未配 | 任务失败，任务中心提示服务不可用 |

---

## 规则

- 停用的功能，进行中的任务如何处理由产品定（一般跑完，不可新建）
- API Key 不在前台展示，仅后台可见

---

## 3. 场景预设


## 一句话

配置前台每个功能、每个场景下的 **参数面板**：Tab、控件、默认值、子选项、提示词。

## 给谁用

AI 配置人员、运营（懂业务参数的人）。

## 入口

管理后台 → **场景预设**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 功能筛选 | 下拉选择：印花提取、抠图、裂变、文生图… |
| 场景列表 | 该功能下各场景（如裂变：默认、服装、手机壳…） |
| 编辑区 | 场景基础信息 + 表单控件树 |
| 预览区 | 模拟前台表单 + 合成提示词预览 |

---

## 可配置内容

| 配置项 | 说明 |
|--------|------|
| 场景 key / 显示名 | 对应前台 Tab 文案 |
| 场景基础提示词 | 拼进最终 prompt 的固定句 |
| 表单控件 | 单选、下拉、滑条、勾选、文本、出图数量按钮 |
| 控件默认值 | 用户打开弹窗时的初始值 |
| 选项子控件 | 选某选项后才出现的额外控件 |
| 控件 promptFragment | 带 `{{value}}` 的片段，参与合成提示词 |
| 启用 / 停用 | 下线整个场景 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建场景** | 增加新 Tab 场景 |
| **编辑** | 打开场景编辑器 |
| **删除** | 删除场景（需确认） |
| **新增控件** | 加一种控件类型 |
| **展开选项 → 子控件** | 给某单选/下拉选项挂子表单 |
| **上移 / 下移** | 调整控件顺序 |
| **保存** | 前台立即生效（用户刷新或重开弹窗） |
| **右侧预览** | 改参数时实时看前台效果与合成 prompt |

---

## 各功能典型场景（参考）

| 功能 | 场景 key 示例 |
|------|----------------|
| 印花提取、抠图、文生图、矢量、侵权 | 默认 |
| 图裂变 | 默认、服装/纺织、手机壳、铁艺、挂钟… |
| 商品套图 | 印花处理 |
| 视频 | model、product、wind |

---

## 对前台的影响

| 改动 | 用户感知 |
|------|----------|
| 新增场景 Tab | 弹窗顶部多一个 Tab |
| 改默认值 | 打开即新默认，不必手调 |
| 加子控件 | 选某选项后多出现设置项 |
| 停用场景 | Tab 消失 |
| 改 prompt | 同样参数生成效果可能变化 |

---

## 规则

- 控件 **类型固定**（radio/select/slider/checkbox/text/textarea/number-buttons），不可自定义新类型（除非产品扩展）
- 场景 key 与前台代码约定一致（如视频 model/product/wind）
- 保存前建议在预览区走一遍主路径

## 4. 推荐案例


## 一句话

配置前台首页「推荐玩法 / 案例」卡片：封面、文案、点进去去哪。

## 给谁用

运营、市场。

## 入口

管理后台 → **推荐案例**

---

## 页面长什么样

案例列表，每条包含：

| 字段 | 说明 |
|------|------|
| 标题 | 卡片主标题 |
| 描述 | 一两句玩法说明 |
| 封面 / 示例图 / 视频 | 视觉素材 |
| 标签 | 高转化、节日、爆款等 |
| 跳转目标 | 某作图功能 / 某工作流模板 / 外链 |
| 状态 | 草稿 / 已发布 / 已下线 |
| 排序 | 数字越小越靠前 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建案例** | 打开表单，填上述字段 |
| **编辑** | 修改已有案例 |
| **发布** | 前台首页展示 |
| **下线** | 前台不再展示 |
| **排序** | 拖拽或填序号，决定卡片顺序 |
| **预览** | 看卡片在前台的长什么样 |

---

## 对前台的影响

| 操作 | 用户看到 |
|------|----------|
| 发布 | 首页多一张案例卡片 |
| 下线 | 卡片消失 |
| 改跳转 | 点击卡片去不同功能/模板 |
| 改排序 | 首页卡片顺序变化 |

---

## 规则

- 未发布案例仅后台可见
- 跳转目标失效时（如模板已下线）应提示运营修复

---

## 5. 工作流模板


## 一句话

制作并下发「官方工作流模板」，用户前台一键复用整套流程。

## 给谁用

运营、高级配置人员。

## 入口

管理后台 → **工作流模板**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 模板列表 | 名称、说明、封面、步骤数、状态 |
| 编辑器 | 与前台类似的工作流画布：节点 + 连线 + 固化参数 |

每条模板状态：**草稿** / **已发布** / **停用**

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建模板** | 空白画布 |
| **编辑** | 改节点、连线、各步默认参数 |
| **发布** | 前台「官方模板」可见 |
| **停用 / 下线** | 前台不可选或隐藏 |
| **预览** | 查看流程结构 |
| **复制** | 基于已有模板复制一份改 |

---

## 配置时要注意什么

- 节点引用的 AI 功能若被 **AI 功能配置** 关闭，模板应标记异常或禁止发布
- 各 AI 节点的默认参数应与 **场景预设** 一致或 intentionally 覆盖
- 发布前在测试账号跑通一遍

---

## 对前台的影响

| 操作 | 用户看到 |
|------|----------|
| 发布新模板 | 模板库多一张官方卡片 |
| 编辑已发布 | 用户「从模板创建」得到新流程结构 |
| 下线 | 模板消失；首页若引用需同步处理 |

---

## 与其他模块的关系

- **推荐案例** 可跳转到某工作流模板
- **任务监控** 可看由该模板产生的任务量

## 6. 套图模板


## 一句话

管理「商品套图」用的样机底图：品类、贴图区域、缩略图。

## 给谁用

运营、设计支持。

## 入口

管理后台 → **套图模板**

---

## 页面长什么样

模板卡片 / 列表：

| 字段 | 说明 |
|------|------|
| 名称 | 如「男士黑色 T 恤白底」 |
| 品类 | T 恤、手机壳、挂钟… |
| 缩略图 | 列表展示用 |
| 样机图 | 用户最终看到的底图 |
| 蒙版 / 贴图区域 | 定义印花贴在哪、多大 |
| 状态 | 启用 / 停用 |
| 排序 | 前台选择器顺序 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新增模板** | 上传样机、蒙版、缩略图，填品类与名称 |
| **编辑** | 更换素材或调整贴图区域 |
| **启用 / 停用** | 控制前台是否可选 |
| **排序** | 调整前台展示顺序 |
| **预览** | 看贴图效果示意 |

---

## 对前台的影响

| 操作 | 用户看到 |
|------|----------|
| 新增 | 套图第一步多一个可选模板 |
| 停用 | 模板从列表消失 |
| 改贴图区域 | 合成时印花位置/大小变化 |

---

## 规则

- 停用不删除历史任务引用，但新建不可选
- 品类 Tab 与模板品类字段对应，用于前台筛选

## 与场景预设的关系

- **套图模板**：第一步选哪个样机
- **场景预设 · 印花处理**：第二步怎么处理印花

## 7. 用户账号


## 一句话

创建前台账号、停用/启用、重置密码，（可选）限制演示账号能力。

## 给谁用

管理员、运营支持。

## 入口

管理后台 → **用户账号**

---

## 页面长什么样

用户列表：

| 列 | 内容 |
|----|------|
| 账号 / 邮箱 | 登录名 |
| 类型 | 普通 / 演示（若有） |
| 状态 | 正常 / 停用 |
| 创建时间 | — |
| 备注 | 运营备注 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建用户** | 填账号、初始密码、类型 → 用户可登录前台 |
| **编辑** | 改备注、类型等 |
| **停用** | 用户无法登录或使用中被迫退出 |
| **启用** | 恢复登录 |
| **重置密码** | 生成或设置新密码，通知用户 |
| **查看任务**（可选） | 跳转 **任务监控** 筛该用户 |

---

## 对前台的影响

| 操作 | 用户经历 |
|------|----------|
| 停用 | 登录提示「账号已停用」 |
| 重置密码 | 旧密码失效 |
| 演示账号限制（若做） | 部分功能或次数受限 |

---

## 规则

- 停用不自动删 **我的空间** 数据
- 初始密码建议首次登录强制修改（若产品支持）

## 8. 任务监控


## 一句话

全站任务的后台视角：谁在跑、卡在哪、为什么失败，必要时代为重试或取消。

## 给谁用

管理员、客服、运维。

## 入口

管理后台 → **任务监控**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 筛选栏 | 功能类型、状态、时间、用户 |
| 任务列表 | 任务 ID、用户、类型、状态、耗时、创建时间 |
| 详情抽屉/页 | 输入图、参数摘要、输出、错误栈（内部）、日志 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **查看详情** | 打开完整任务信息 |
| **重试** | 用相同输入+参数重新提交（策略允许时） |
| **取消 / 废弃** | 终止排队或卡死任务 |
| **导出**（可选） | 导出筛选结果供分析 |

---

## 对前台的影响

| 后台操作 | 用户任务中心 |
|----------|--------------|
| 重试成功 | 状态变成功，出现新结果 |
| 取消 | 状态变已取消 |
| 仅查看 | 无变化 |

---

## 规则

- 重试 / 取消是否开放给客服由权限控制
- 失败原因对用户展示用人话，详情页可看技术原因（仅后台）
- 与 **概览** 联动：失败率异常时概览预警

---

## 典型排查路径

1. 用户反馈「抠图一直失败」→ 筛选该用户 + 抠图 + 失败  
2. 看详情：是否 Key 失效、图片过大、模型超时  
3. 修 **AI 功能配置** 或通知用户换图  
4. 必要时 **重试**

## 9. 系统设置


## 一句话

平台级全局配置：存储、安全、维护模式等，不归属单个 AI 功能。

## 给谁用

超级管理员、运维。

## 入口

管理后台 → **系统设置**

---

## 页面长什么样（常见分组）

| 分组 | 典型配置项 |
|------|------------|
| 安全 | 管理后台访问策略、会话时长 |
| 存储 | 文件存储位置、CDN、上传大小上限 |
| 平台开关 | 维护模式、灰度功能开关 |
| 通知 | 邮件/Webhook 告警（可选） |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **保存** | 全局配置生效 |
| **测试连接** | 检查存储等第三方是否可用，成功/失败提示 |

---

## 对前台的影响

| 配置 | 用户可能遇到 |
|------|--------------|
| 上传大小调小 | 大文件上传失败，提示超限 |
| 存储故障 | 上传、预览、下载失败 |
| **维护模式** 开启 | 全站或部分功能提示「系统维护中」 |
| 下载链接有效期 | 下载中心链接过期需重新生成 |

---

## 规则

- 维护模式开启前应有运营通知
- 敏感配置（密钥）仅管理员可见
- 修改存储配置不迁移历史文件时需有迁移方案（运维流程，非本页范围）

---

## 与其他模块的关系

- 影响 **我的空间** 上传、**下载中心** 链接、**任务中心** 结果访问
- 不影响 **场景预设** 业务参数
