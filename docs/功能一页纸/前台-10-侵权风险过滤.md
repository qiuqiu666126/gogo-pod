# 前台 · 侵权风险过滤

## 一句话

检测图片是否可能侵犯商标、IP 或含敏感元素，给出风险等级与上架建议。

## 解决什么问题

POD 卖家批量铺货时避免侵权下架、账号风险。

## 入口

左侧菜单 → **作图** → **侵权风险过滤**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 待检测图片 |
| 提示条 | 建议先「图案裁切」再检测（特殊构图时） |
| 模式选择 | 深度过滤 / 基础过滤 |
| 合规勾选 | 「侵权风险过滤功能使用须知」 |
| 底部 | 取消、提交 |

---

## 典型参数

| 模式 | 用户理解 |
|------|----------|
| **深度过滤** | 结合 TRO 案件、艺术家版权库、全网检索，出证据链式报告 |
| **基础过滤** | 快速比对知名商标与热门 IP |

---

## 结果页用户看到什么

| 信息 | 说明 |
|------|------|
| 风险等级 | 低 / 中 / 高 |
| 命中原因 | 大白话说明（如疑似某品牌 logo） |
| 建议 | 可用 / 建议复核 / 不建议上架 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **示例**（提示条内） | 查看何种图建议先裁切 |
| **提交** | 创建检测任务（建议需勾选须知） |
| 结果 **复制报告** | 复制结论供运营留存 |

---

## 规则（重要）

- **高风险** 建议：默认 **不允许直接进入上架流程**，或必须二次确认
- 中风险：提示复核，用户自行决定
- 低 risk：可正常后续作图/上架
- 深度模式耗时可能更长，应在任务中心展示预计时间（若支持）

---

## 与其他模块的关系

- 检测通过/复核后 → **商品库** / **上架**
- 与 **印花提取** 组合：先提取图案再检测更准

## 后台配置影响

AI 功能开关 + 场景预设（模式选项）
