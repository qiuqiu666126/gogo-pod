# 管理后台 · 场景预设

## 一句话

配置前台每个功能、每个场景下的 **参数面板**：Tab、控件、默认值、子选项、提示词。

## 给谁用

AI 配置人员、运营（懂业务参数的人）。

## 入口

管理后台 → **场景预设**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 功能筛选 | 下拉选择：印花提取、抠图、裂变、文生图… |
| 场景列表 | 该功能下各场景（如裂变：默认、服装、手机壳…） |
| 编辑区 | 场景基础信息 + 表单控件树 |
| 预览区 | 模拟前台表单 + 合成提示词预览 |

---

## 可配置内容

| 配置项 | 说明 |
|--------|------|
| 场景 key / 显示名 | 对应前台 Tab 文案 |
| 场景基础提示词 | 拼进最终 prompt 的固定句 |
| 表单控件 | 单选、下拉、滑条、勾选、文本、出图数量按钮 |
| 控件默认值 | 用户打开弹窗时的初始值 |
| 选项子控件 | 选某选项后才出现的额外控件 |
| 控件 promptFragment | 带 `{{value}}` 的片段，参与合成提示词 |
| 启用 / 停用 | 下线整个场景 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建场景** | 增加新 Tab 场景 |
| **编辑** | 打开场景编辑器 |
| **删除** | 删除场景（需确认） |
| **新增控件** | 加一种控件类型 |
| **展开选项 → 子控件** | 给某单选/下拉选项挂子表单 |
| **上移 / 下移** | 调整控件顺序 |
| **保存** | 前台立即生效（用户刷新或重开弹窗） |
| **右侧预览** | 改参数时实时看前台效果与合成 prompt |

---

## 各功能典型场景（参考）

| 功能 | 场景 key 示例 |
|------|----------------|
| 印花提取、抠图、文生图、矢量、侵权 | 默认 |
| 图裂变 | 默认、服装/纺织、手机壳、铁艺、挂钟… |
| 商品套图 | 印花处理 |
| 视频 | model、product、wind |

---

## 对前台的影响

| 改动 | 用户感知 |
|------|----------|
| 新增场景 Tab | 弹窗顶部多一个 Tab |
| 改默认值 | 打开即新默认，不必手调 |
| 加子控件 | 选某选项后多出现设置项 |
| 停用场景 | Tab 消失 |
| 改 prompt | 同样参数生成效果可能变化 |

---

## 规则

- 控件 **类型固定**（radio/select/slider/checkbox/text/textarea/number-buttons），不可自定义新类型（除非产品扩展）
- 场景 key 与前台代码约定一致（如视频 model/product/wind）
- 保存前建议在预览区走一遍主路径
