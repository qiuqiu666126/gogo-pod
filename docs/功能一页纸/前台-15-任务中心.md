# 前台 · 任务中心

## 一句话

所有 AI 与工作流任务的进度、结果、失败原因和后续操作的统一入口。

## 解决什么问题

任务异步执行，用户需要知道「跑完没有、失败为什么、结果在哪」。

## 入口

左侧菜单 → **任务中心**

---

## 页面长什么样

### 列表页

| 列 | 内容 |
|----|------|
| 任务类型 | 抠图、裂变、套图、工作流、视频… |
| 状态 | 排队中 / 处理中 / 成功 / 失败 / 已取消 |
| 创建时间 | — |
| 摘要 | 张数、场景名等 |

**筛选：** 类型、状态、时间范围

### 详情页

| 区域 | 内容 |
|------|------|
| 输入图 | 本次用的素材 |
| 参数摘要 | 用人话展示（如「模式：文字强化，出图 4 张」） |
| 输出结果 | 网格预览，可点大图 |
| 失败信息 | 原因 + 是否可重试 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 点击列表行 | 进入详情 |
| **查看结果 / 对比** | 输入 vs 输出对比视图 |
| **入库** | 输出存到 **我的空间**（或商品库） |
| **下载** | 单个或打包下载 → **下载中心** 有记录 |
| **重试** | 用相同输入+参数再跑一次（失败且允许时） |
| **再创作** | 用输出作为新任务输入（如裂变 → 套图） |
| **取消**（若支持） | 排队中任务取消 |

---

## 规则

| 状态 | 用户能做什么 |
|------|--------------|
| 排队中 | 可取消（若支持） |
| 处理中 | 仅查看，不可改参数 |
| 成功 | 入库、下载、再创作 |
| 失败 | 看重试；后台关闭的功能不可重试并说明原因 |
| 已取消 | 可重新从原模块发起 |

---

## 与其他模块的关系

所有作图、视频、工作流任务汇聚于此。

## 后台配置影响

| 操作 | 影响 |
|------|------|
| 场景预设变更 | 仅影响 **新任务** 默认值，不改变历史任务展示 |
| 功能停用 | 重试按钮不可用 |
| 任务监控 · 后台重试/取消 | 用户侧状态同步更新 |
