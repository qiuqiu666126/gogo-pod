# 前台 · 视频生成

## 一句话

把静态商品图生成短视频，用于主图视频、广告素材。

## 解决什么问题

平台 increasingly 需要动态展示；实拍视频成本高。

## 入口

左侧菜单 → **视频**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| **场景 Tab** | 模特动作 / 商品律动 / 风铃转动 |
| 素材上传 | 参考图（风铃场景规则可能不同） |
| 参数区 | 创意方式、预设、生成模式、时长、数量等 |
| 底部 | 取消、提交 |

---

## 分场景说明

### 模特动作

| 元素 | 说明 |
|------|------|
| 提示条 | 适用模特上身图；纯衣服图建议先去「模特试衣」 |
| 视频创意 | **效果预设**（自然动态、休闲走动、杂志写真等）或 **自定义提示词** |
| 生成模式 | 标准（快、省）/ 高清（画质好） |

**交互：** 选「自定义提示词」→ 出现多行文本框，可增删多条描述。

### 商品律动

| 元素 | 说明 |
|------|------|
| 视频创意 | 效果预设（宠物用品、地垫、毛毯、手机壳、桌布等）或自定义 |
| 生成模式 | 标准 / 高清 |
| 视频时长 | 5s / 10s |
| 生成数量 | 1 / 2 |

### 风铃转动

| 元素 | 说明 |
|------|------|
| 风铃样式 | 花形 / 爱心 / 圣诞树等 |
| 背景 | 选择背景样式 |
| 视频时长 | 5s / 10s |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 场景 **Tab** | 切换场景，参数区整体切换 |
| **效果预设** 卡片/选项 | 选中一种动作风格 |
| **+ 新增提示词** | 自定义模式下增加输入行 |
| **提交** | 创建视频任务 → **任务中心** |
| 结果 **播放预览** | 站内播放 |
| **入库 / 下载** | 与其他模块一致 |

---

## 规则

- 不同场景参数集不同，由 **场景预设** 分 scene 配置
- 自定义 prompt 为空时不应提交
- 视频生成耗时较长，任务中心应显示「处理中」

## 后台配置影响

| 配置 | 影响 |
|------|------|
| 场景预设 · model/product/wind | 各 Tab 参数与预设列表 |
| AI 功能配置 | 视频模块开闭 |
