# 前台 · 印花提取

## 一句话

从商品实拍图里把印花/图案单独抠出来，常见输出透明底 PNG。

## 解决什么问题

商品图上有衣服、背景等干扰，卖家只需要中间的图案去套图或再创作。

## 入口

左侧菜单 → **作图** → **印花图提取**（或从找图/我的空间「发送到作图」带入图片）

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 上传 / 从我的空间选 / 拖拽 |
| 参数区 | 由后台「场景预设」配置（见下表典型项） |
| 底部 | 取消、**提交** |

---

## 典型参数（以场景预设为准）

| 参数 | 用户理解 | 交互 |
|------|----------|------|
| 模式选择 | 专项提取 vs 全能提取 | 单选；专项适合遮挡少的印花类，全能适合褶皱、低清、全副印花 |
| 透明底图 | 输出是否透明背景 | 开关 |
| 分辨率 | 标清(1k) / 超清(4k) | 单选 |
| 尺寸比例 | 1:1、2:3 等 | 下拉 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **取消** | 关闭弹窗，不提交 |
| **提交** | 创建任务 → **任务中心** 看进度；完成后可预览结果 |
| **入库**（结果页） | 存到 **我的空间** |
| **下载** | 打包下载，记录进 **下载中心** |
| **再创作** | 用输出图作为输入跑其他功能（如套图） |
| **对比查看** | 原图 vs 提取结果 |
| **重置参数** | 恢复后台配置的默认值 |

---

## 规则

- 未上传素材时 **提交** 应禁用或提示
- 参数默认值来自后台 **场景预设 → 印花图提取 → 默认**
- 后台关闭本功能 → 入口隐藏或置灰

---

## 典型后续流程

提取出的印花 → **商品套图** → 入库 / 上架

## 后台配置影响

| 配置项 | 前台变化 |
|--------|----------|
| AI 功能配置 · 停用 | 入口不可用 |
| 场景预设 · 控件/默认值 | 参数项、默认值、提示词效果 |
