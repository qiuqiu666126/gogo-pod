# 前台 · 我的空间

## 一句话

用户所有素材的「图库」：上传、分类、再发送到作图或工作流。

## 解决什么问题

作图结果、采集图、本地上传分散在各处，需要统一资产管理。

## 入口

左侧菜单 → **我的空间**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 工具栏 | 上传、新建标签、批量选择、搜索 |
| 标签侧栏 / 筛选 | 按标签、类型（图/视频/矢量） |
| 网格图库 | 缩略图、文件名、类型图标 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **上传** | 选择本地文件上传；完成后出现在网格 |
| **新建标签** | 创建标签名，可给素材打标签 |
| **批量选择** | 进入多选模式 |
| **发送到作图** | 选功能类型，带着选中素材打开对应作图弹窗 |
| **发送到工作流** | 带着选中素材打开工作流运行 |
| **下载** | 下载选中文件 |
| **删除** | 删除选中素材（需确认） |
| 单击素材 | 预览大图 / 详情侧栏 |
| 双击 / **详情** | 查看来源、尺寸、创建时间、关联任务 |

---

## 规则

- 删除前确认；若素材已被商品引用应提示
- 上传大小、格式限制由 **系统设置** 决定（超限提示）
- 容量接近上限时提示（若产品有配额）

---

## 与其他模块的关系

| 模块 | 关系 |
|------|------|
| 找图 / 任务中心 | 入库来源 |
| 作图 / 工作流 | 输出目的地 & 输入来源 |
| 商品库 | 选图创建商品 |

## 后台配置影响

系统设置：上传大小、存储、容量展示
