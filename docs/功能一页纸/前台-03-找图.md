# 前台 · 找图

## 一句话

从外部电商网站采集图片到平台，或用一张图去外部网站搜相似图。

## 解决什么问题

POD 卖家素材来源在外部平台，需要先把图「抓进来」再作图。

## 入口

左侧菜单 → **找图**

---

## 页面长什么样

### A. 插件采集（采集任务列表）

| 列 / 信息 | 说明 |
|-----------|------|
| 采集来源 | 网站 / 店铺 |
| 采集时间 | — |
| 图片数量 | 本次采了多少张 |
| 状态 | 同步中 / 完成 / 失败 |

### B. 以图搜图

| 区域 | 说明 |
|------|------|
| 上传 / 选图 | 本地或从我的空间 |
| 外部站点按钮 | Google、Pinterest、Shutterstock 等 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **查看详情** | 展开该次采集的全部图片 |
| **批量选择** | 进入多选模式 |
| **入库到我的空间** | 选中图保存到 **我的空间** |
| **发送到作图** | 带着选中图打开 **作图** 模块（需再选具体功能） |
| **发送到工作流** | 选中图作为工作流输入 |
| **去 Google / Pinterest / … 搜** | **新开浏览器标签页**，跳转外部站以图搜图（不在站内完成） |

---

## 规则

- 采集任务失败应显示原因，支持重试（若产品支持）
- 以图搜图依赖外部网站，站内只负责跳转
- 发送到作图 / 工作流时，若未选图应提示先选择

---

## 与其他模块的关系

| 模块 | 关系 |
|------|------|
| 我的空间 | 采集结果入库 |
| 作图 / 工作流 | 采集图作为输入 |

## 后台配置影响

一般无强依赖；若未来做「禁止某些外部站点入口」，由 **系统设置** 控制。
