# 前台 · 上架

## 一句话

把作图结果整理成「商品」，绑定店铺，按平台模板导出刊登包。

## 解决什么问题

作图完成后还要填标题、选平台格式、打包下载才能上架，需要一站式出口。

## 入口

左侧菜单 → **上架**（含商品库、店铺管理、刊登模板子页）

---

## 子功能一：商品库

### 页面长什么样

| 区域 | 内容 |
|------|------|
| 商品列表 | 主图缩略图、标题、状态 |
| 筛选 / 搜索 | 按状态、标签等 |

### 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建商品** | 从我的空间 / 任务结果选图，填标题、描述、标签 |
| **编辑** | 改标题、换主图、增删套图、改标签 |
| **删除** | 删除商品（需确认） |
| **导出** | 按选定平台模板导出 |

### 常见字段

标题、描述、主图、套图多图、标签/关键词、类目（视平台）

---

## 子功能二：店铺管理

| 按钮 | 点击后发生什么 |
|------|----------------|
| **绑定店铺** | 填写平台授权信息（本期可占位流程） |
| **解绑** | 解除关联，需确认 |

---

## 子功能三：刊登模板

**流程：**

1. 选择平台（Temu / Amazon / Shein …）
2. 选择该平台的 **刊登模板**
3. 选择要导出的商品
4. 点 **导出刊登包**

**导出结果：** 压缩包，内含图片 + 表格或字段文件（字段由后台模板定义）

---

## 规则

- 必填字段未填时导出应拦截并提示缺什么
- **侵权高风险** 商品导出前应警告或拦截（与侵权过滤联动）
- 刊登模板字段、命名规则由后台 **刊登模板 / 套图模板** 配置

---

## 与其他模块的关系

| 来源 | 关系 |
|------|------|
| 任务中心 / 我的空间 | 选图创建商品 |
| 标题提取 | 填充标题 |
| 下载中心 | 导出大包记录 |

## 后台配置影响

平台列表、刊登模板字段、导出规则、套图模板
