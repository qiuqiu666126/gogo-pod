# 前台 · 首页与工作流

## 一句话

把多个 AI 步骤串成一条流水线，选一批图一键跑完（如抠图 → 裂变 → 套图）。

## 解决什么问题

避免在 8 个作图功能之间来回切换；适合批量、标准化处理。

## 入口

左侧菜单 → **首页 / 工作流**

---

## 页面长什么样

### 工作流首页

| 区域 | 内容 |
|------|------|
| 欢迎 / 快速开始 | 引导新建或从模板开始 |
| 最近任务 | 最近跑过的工作流及状态 |
| 推荐玩法 / 案例 | 运营配置的示例卡片（来自后台「推荐案例」） |

### 工作流编辑器（画布）

| 区域 | 内容 |
|------|------|
| 画布 | 节点与连线 |
| 节点面板 | 可添加的节点类型列表 |
| 右侧参数面板 | 选中 AI 节点后出现，调该步参数 |
| 顶部操作栏 | 保存、运行、返回等 |

### 模板库

| 分类 | 说明 |
|------|------|
| 官方模板 | 后台下发，全员可见（已发布） |
| 我的模板 | 用户自己保存的流程 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建工作流** | 打开空白画布 |
| **从模板创建** | 选模板 → 生成可编辑的工作流副本 |
| **添加节点** | 节点出现在画布上 |
| **连线** | 拖拽连接两节点，定义执行顺序 |
| **删除节点 / 连线** | 该步骤不再执行 |
| **保存** | 保存当前流程，下次可继续编辑 |
| **运行 / 开始** | 选择输入图片 → 提交任务 → 去 **任务中心** 看进度 |
| **另存为模板**（若开放） | 存为「我的模板」 |
| **使用 / 创建**（模板卡片） | 用该模板生成新工作流 |
| **预览**（模板卡片） | 查看模板包含哪些步骤 |

---

## 规则

- 流程必须能从「输入」走到「输出」，否则运行前应提示不完整
- 后台停用某 AI 功能 → 画布中对应 **节点不可选或置灰**，已有流程中含该节点时运行应提示
- 运行中的任务在 **任务中心** 查看；单步失败可看到是哪一步、什么原因

---

## 与其他模块的关系

| 模块 | 关系 |
|------|------|
| 我的空间 / 找图 | 可作为工作流输入来源 |
| 作图 | 工作流中的 AI 节点对应各作图能力 |
| 任务中心 | 运行结果、进度、重试 |
| 下载中心 | 若输出节点配置为下载 |

## 后台配置影响

| 后台模块 | 影响 |
|----------|------|
| AI 功能配置 | 某功能关闭 → 节点不可用 |
| 场景预设 | 节点内参数面板与单功能作图一致 |
| 工作流模板 | 官方模板增删改 |
| 推荐案例 | 首页案例卡片 |
