# 前台 · 商品套图

## 一句话

把印花合成到商品样机底图上，一步生成可上架的多角度/多场景商品图。

## 解决什么问题

卖家有图案但没有商品实拍，需要快速出「穿在身上/放在桌上」的效果图。

## 入口

左侧菜单 → **作图** → **商品套图**

---

## 页面长什么样（两步向导）

### 第一步 · 模板选择

| 区域 | 内容 |
|------|------|
| 品类 Tab | 推荐、服饰、铁皮画、家用纺织、挂钟、手机壳等 |
| 模板卡片网格 | 缩略图、名称、「官方」标签、多选框 |
| 底部 | 取消、**下一步** |

### 第二步 · 印花选择

| 区域 | 内容 |
|------|------|
| 添加素材 | 上传 / 从我的空间选 |
| 印花处理参数 | 查重、边缘、描边、压缩、格式等（场景预设） |
| 底部 | 取消、**提交** |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| 品类 **Tab** | 筛选模板列表 |
| 点击模板卡片 | 勾选 / 取消勾选（可多选） |
| **下一步** | 进入第二步（可从我的空间带入素材时直接进第二步） |
| **提交** | 对选中模板 + 印花创建套图任务 |
| **取消** | 关闭向导 |

---

## 典型参数（第二步 · 印花处理）

| 参数 | 说明 |
|------|------|
| 印花查重校验 | 是否检测重复印花 |
| 边缘处理 | 裁掉边缘空白 |
| 描边 | 开关；可选智能/强制、颜色、粗细 |
| 智能反转 | 自动反转颜色适配底图 |
| 压缩图片 | 限制输出文件大小 |
| 文件格式 | JPEG / PNG / WEBP |

**交互规则：** 开启「描边」才出现描边方式；选「强制描边」才出现颜色选项。

---

## 规则

- 第一步至少选 1 个模板才能下一步（建议）
- 第二步至少 1 张印花才能提交
- 模板列表来自后台 **套图模板**；停用的模板不展示
- 印花处理参数来自 **场景预设 → 商品套图 → 印花处理**

---

## 典型后续流程

套图结果 → **商品库** / **下载** / **上架导出**

## 后台配置影响

| 模块 | 影响 |
|------|------|
| 套图模板 | 可选样机增删、排序 |
| 场景预设 | 第二步参数面板 |
| AI 功能配置 | 功能开闭 |
