# 管理后台 · 套图模板

## 一句话

管理「商品套图」用的样机底图：品类、贴图区域、缩略图。

## 给谁用

运营、设计支持。

## 入口

管理后台 → **套图模板**

---

## 页面长什么样

模板卡片 / 列表：

| 字段 | 说明 |
|------|------|
| 名称 | 如「男士黑色 T 恤白底」 |
| 品类 | T 恤、手机壳、挂钟… |
| 缩略图 | 列表展示用 |
| 样机图 | 用户最终看到的底图 |
| 蒙版 / 贴图区域 | 定义印花贴在哪、多大 |
| 状态 | 启用 / 停用 |
| 排序 | 前台选择器顺序 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新增模板** | 上传样机、蒙版、缩略图，填品类与名称 |
| **编辑** | 更换素材或调整贴图区域 |
| **启用 / 停用** | 控制前台是否可选 |
| **排序** | 调整前台展示顺序 |
| **预览** | 看贴图效果示意 |

---

## 对前台的影响

| 操作 | 用户看到 |
|------|----------|
| 新增 | 套图第一步多一个可选模板 |
| 停用 | 模板从列表消失 |
| 改贴图区域 | 合成时印花位置/大小变化 |

---

## 规则

- 停用不删除历史任务引用，但新建不可选
- 品类 Tab 与模板品类字段对应，用于前台筛选

## 与场景预设的关系

- **套图模板**：第一步选哪个样机
- **场景预设 · 印花处理**：第二步怎么处理印花
