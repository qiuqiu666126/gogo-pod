# 前台 · 标题提取

## 一句话

从图片中 OCR 提取文字，或根据画面生成适合电商的标题与关键词。

## 解决什么问题

批量上图时缺标题、缺搜索词，手工写效率低。

## 入口

左侧菜单 → **作图** → **标题提取**（或商品库内「从图生成标题」）

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 素材上传区 | 单张或多张图 |
| 结果区 | 提取/生成的标题、关键词列表 |
| 操作 | 复制、写入商品库 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **提交** | 创建提取任务 |
| **复制标题** | 复制到剪贴板 |
| **复制全部** | 复制标题+关键词 |
| **写入商品库**（若有） | 填入当前编辑商品的标题字段 |
| **重新生成** | 同一图再跑一次（换表述） |

---

## 规则

- 图中无清晰文字时，OCR 结果可能为空，应提示改用语义生成标题
- 生成标题需符合目标平台字数限制（可在结果处显示字数）

---

## 与其他模块的关系

标题 → **商品库** → **上架 / 刊登模板导出**

## 后台配置影响

AI 功能开关；若未来有「标题风格」参数，由场景预设配置
