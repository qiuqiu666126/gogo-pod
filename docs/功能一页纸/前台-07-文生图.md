# 前台 · 文生图

## 一句话

输入文字描述（提示词），AI 直接生成设计图；支持多条 prompt 批量出图。

## 解决什么问题

没有现成素材时，从创意文字快速出稿。

## 入口

左侧菜单 → **作图** → **文生图**

---

## 页面长什么样

| 区域 | 内容 |
|------|------|
| 添加方式 | 图反推提示词、提示词助手、导入 Excel 等 |
| 模式 / 参数区 | 模式、比例、出图数量（来自场景预设） |
| 提示词列表 | 多条 prompt，可勾选、增删 |
| 底部 | 取消、提交 |

---

## 典型参数

| 参数 | 说明 |
|------|------|
| 模式选择 | 艺术设计 / 文字强化 / IP 增强 / 四方连续图 |
| 尺寸比例 | 1:1、2:3 等 |
| 出图数量 | 通常 2 或 4 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **图反推提示词** | 上传参考图 → 自动生成 prompt 填入列表 |
| **提示词助手** | 打开辅助写词（跳转或侧栏） |
| **导入 EXCEL** | 批量导入多条 prompt |
| **+ 添加新的提示词** | 列表增加一行输入框 |
| 行内 **删除** | 移除该条 prompt |
| 行内 **勾选** | 控制该条是否参与提交 |
| **提交** | 对「已勾选且有内容」的 prompt 批量生成 |

---

## 规则

- 建议提示使用英文（页面可有说明文案）
- 至少 1 条有效 prompt 才能提交
- 出图数量 × 有效 prompt 条数 = 总任务量（按产品设计计费/限额）
- 提示词列表为前台专用 UI，模式/比例/数量由 **场景预设** 配置

---

## 典型后续流程

生成图 → **入库** → **套图** / **上架**

## 后台配置影响

场景预设（模式、比例、数量默认值）+ AI 功能开关
