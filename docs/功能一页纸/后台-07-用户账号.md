# 管理后台 · 用户账号

## 一句话

创建前台账号、停用/启用、重置密码，（可选）限制演示账号能力。

## 给谁用

管理员、运营支持。

## 入口

管理后台 → **用户账号**

---

## 页面长什么样

用户列表：

| 列 | 内容 |
|----|------|
| 账号 / 邮箱 | 登录名 |
| 类型 | 普通 / 演示（若有） |
| 状态 | 正常 / 停用 |
| 创建时间 | — |
| 备注 | 运营备注 |

---

## 按钮与交互

| 按钮 | 点击后发生什么 |
|------|----------------|
| **新建用户** | 填账号、初始密码、类型 → 用户可登录前台 |
| **编辑** | 改备注、类型等 |
| **停用** | 用户无法登录或使用中被迫退出 |
| **启用** | 恢复登录 |
| **重置密码** | 生成或设置新密码，通知用户 |
| **查看任务**（可选） | 跳转 **任务监控** 筛该用户 |

---

## 对前台的影响

| 操作 | 用户经历 |
|------|----------|
| 停用 | 登录提示「账号已停用」 |
| 重置密码 | 旧密码失效 |
| 演示账号限制（若做） | 部分功能或次数受限 |

---

## 规则

- 停用不自动删 **我的空间** 数据
- 初始密码建议首次登录强制修改（若产品支持）
